WedLove - Wedding Announcement - React Template

Getting Started

WedLove – is a clean, unique, responsive React wedding template. It is a modern, beautiful, elegant wedding template. We have tried to use modern tools like React, Bootstrap, for building this template. Couple details, Love story, Countdown, Groomsmen and Bridesmaids, Events almost all kinds of wedding features we have tried to put into it. RSVP form can send an email. It has a unique design with amazing features for couples who want to build their wedding website. It looks great on mobile devices and desktops. It is clean and simple.

It’s a fully responsive design that will make your site look great on all kinds of modern devices including mobile phones, laptops, tablets, etc. All the sections are neatly placed on the homepage to make sure that it looks minimal and distraction-free.

installing

Install Node.Js
  • First, you download node.js and install.
  • you can download node js from nodejs.org
  • (1) Open CMD and set path on your system Drive:\WedLove>
  • (2) Run command Drive:\WedLove> npm i
  • (3) Run command Drive:\WedLove> npm start

Animation

  • import WOW from "wowjs";
  • Set Configuration:
        const [wow, setWow] = useState(new WOW.WOW())
        useEffect(() => {
            setWow(wow.init())
        }, [wow]);
  • Set Div className: "wow"  ,  "animation name"  &  "animation config attr"
    Ex:-
        <div className="wow fadeInRight   data-wow-delay="0.5s"   data-wow-duration="1s"   data-wow-offset="10"   data-wow-iteration="10"></div>
  • Animation list:- wow.js
  • Other info wow.js

Set Up Live

  • 1). Open package.json and homepage add your domain url and folder name "homepage": "https://example.com/WedLove"
  • 2). Run command   Drive:\WedLove> npm run build
  • WedLove in side Generate  build folder.
  • Build folder convert zip file & upload your cpanel hosting.
  • Now your site is Live...
